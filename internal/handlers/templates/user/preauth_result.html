Шаг 0 — Подключиться к нашему control server

Наш tailnet работает на собственном headscale, а не на дефолтном Tailscale. Прежде чем вводить ключ — нужно на устройстве указать адрес нашего control server:

{{.ControlURL}}

Ниже — точные шаги для {{.OSLabel}}: сначала включите custom control server, затем введите ключ.

Инструкция для {{.OSLabel}}

{{if eq .OS "android"}}
  1. Установите Tailscale из Google Play (если ещё нет).
  2. Откройте приложение, не нажимайте Sign in. Сразу: меню (слева сверху) → Settings → прокрутите вниз до "Use custom coordination server" → переключите в ON.
  3. В поле "Coordination server URL" вставьте: {{.ControlURL}}
  4. Вернитесь на главный экран → Sign in. Откроется страница нашего headscale.
  5. На странице входа найдите опцию "Use a preauth key" (внизу формы) → нажмите.
  6. Вставьте скопированный выше ключ → Continue.
  7. Готово. Устройство появится на странице устройств через несколько секунд.
Если опции "Use a preauth key" нет в вашей версии Tailscale — обновите приложение из Google Play (нужна версия 1.50+).
{{else if eq .OS "ios"}}
  1. Установите Tailscale из App Store (если ещё нет).
  2. Откройте приложение → Settings → "Use Custom Control Server" → переключите в ON.
  3. В поле "Control Server URL" вставьте: {{.ControlURL}}
  4. Вернитесь на главный экран → Sign in. Откроется страница нашего headscale.
  5. Найдите опцию "Use a preauth key" (внизу формы входа) → нажмите.
  6. Вставьте скопированный выше ключ → Continue.
  7. Готово. Устройство появится на странице устройств через несколько секунд.
Нужен Tailscale 1.50+ (App Store). На более старых — обновите.
{{else if eq .OS "linux"}}

Одна команда — установит Tailscale (если нет), подключится к нашему control server и зарегистрирует устройство по ключу:

curl -fsSL https://tailscale.com/install.sh | sh
sudo tailscale up --login-server={{.ControlURL}} --authkey={{.Key}}

Флаг --accept-routes добавьте если хотите принимать маршруты от других узлов (например, exit nodes). Полный список: tailscale up --help.

Готово — устройство появится на странице устройств через несколько секунд.

{{else if eq .OS "macos"}}

Способ 1 — через GUI (проще):

  1. App Store → установите Tailscale.
  2. Menu bar (правый верх) → кликните на иконку Tailscale → "Use Custom Control Server" → вставьте: {{.ControlURL}}
  3. В том же меню → Sign in → откроется страница нашего headscale.
  4. Найдите "Use a preauth key" → вставьте ключ → Continue.

Способ 2 — через CLI (если стоит Homebrew):

brew install --cask tailscale-app
open -a Tailscale
/usr/local/bin/tailscale up --login-server={{.ControlURL}} --authkey={{.Key}}

Готово — устройство появится на странице устройств через несколько секунд.

{{else if eq .OS "windows"}}

Способ 1 — через GUI (проще):

  1. Скачайте установщик: tailscale.com/download/windows → установите.
  2. System tray (правый нижний угол) → кликните на иконку Tailscale → Sign in → откроется браузер.
  3. В браузере откроется страница нашего headscale. Найдите "Use a preauth key" → вставьте ключ → Continue.

Способ 2 — через PowerShell (надёжнее):

Invoke-WebRequest https://pkgs.tailscale.com/stable/tailscale-setup-latest.exe -OutFile $env:TEMP\ts.exe
Start-Process $env:TEMP\ts.exe -ArgumentList "/quiet" -Wait
& "$env:ProgramFiles\Tailscale\tailscale.exe" up --login-server={{.ControlURL}} --authkey={{.Key}}

Готово — устройство появится на странице устройств через несколько секунд.

{{else}}
Неизвестная ОС "{{.OS}}". Выберите ОС на странице устройств и получите ключ заново.
{{end}}

Хотите войти под ДРУГИМ аккаунтом?

По умолчанию ключ привязан к {{.Username}}. Если на этом устройстве уже выполнен вход в Tailscale под другим аккаунтом — сначала выйдите: 

tailscale logout

Затем повторите команду tailscale up --login-server={{.ControlURL}} --authkey={{.Key}}. На мобильных (Android/iOS) — выйдите через → Sign out, затем повторите шаги выше.
