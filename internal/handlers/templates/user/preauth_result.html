Step 0 — Connect to our control server

Our tailnet runs on a self-hosted headscale, not the default Tailscale control server. Before entering the key, point your device at our control server:

{{.ControlURL}}

Steps for {{.OSLabel}} below: first enable the custom control server, then enter the key.

Instructions for {{.OSLabel}}

{{if eq .OS "android"}}
  1. Install Tailscale from Google Play (if you don't have it).
  2. Open the app — do not tap "Sign in" yet. Instead: menu (top-left) → Settings → scroll down to "Use custom coordination server" → switch ON.
  3. Paste into the "Coordination server URL" field: {{.ControlURL}}
  4. Go back to the main screen → Sign in. Our headscale page will open.
  5. Find the "Use a preauth key" option (at the bottom of the form) → tap it.
  6. Paste the key you copied above → Continue.
  7. Done. The device will appear on the My devices page in a few seconds.
If you don't see "Use a preauth key" in your Tailscale version, update the app (1.50+ required).
{{else if eq .OS "ios"}}
  1. Install Tailscale from the App Store (if you don't have it).
  2. Open the app → Settings → "Use Custom Control Server" → switch ON.
  3. Paste into the "Control Server URL" field: {{.ControlURL}}
  4. Go back to the main screen → Sign in. Our headscale page will open.
  5. Find "Use a preauth key" (at the bottom of the login form) → tap it.
  6. Paste the key → Continue.
  7. Done. The device will appear on the My devices page in a few seconds.
Tailscale 1.50+ required (App Store). On older versions — update first.
{{else if eq .OS "linux"}}

One command — installs Tailscale (if needed), connects to our control server, and registers the device with the key:

curl -fsSL https://tailscale.com/install.sh | sh
sudo tailscale up --login-server={{.ControlURL}} --authkey={{.Key}}

Add --accept-routes if you want to accept routes from other nodes (e.g. exit nodes). Full list: tailscale up --help.

Done — the device will appear on the My devices page in a few seconds.

{{else if eq .OS "macos"}}

Method 1 — GUI (easier):

  1. App Store → install Tailscale.
  2. Menu bar (top right) → click the Tailscale icon → "Use Custom Control Server" → paste: {{.ControlURL}}
  3. Same menu → Sign in → our headscale page opens.
  4. Find "Use a preauth key" → paste the key → Continue.

Method 2 — CLI (if you have Homebrew):

brew install --cask tailscale-app
open -a Tailscale
/usr/local/bin/tailscale up --login-server={{.ControlURL}} --authkey={{.Key}}

Done — the device will appear on the My devices page in a few seconds.

{{else if eq .OS "windows"}}

Method 1 — GUI (easier):

  1. Download the installer: tailscale.com/download/windows → install.
  2. System tray (bottom right) → click the Tailscale icon → Sign in → browser opens.
  3. Our headscale page opens in the browser. Find "Use a preauth key" → paste the key → Continue.

Method 2 — PowerShell (more reliable):

Invoke-WebRequest https://pkgs.tailscale.com/stable/tailscale-setup-latest.exe -OutFile $env:TEMP\ts.exe
Start-Process $env:TEMP\ts.exe -ArgumentList "/quiet" -Wait
& "$env:ProgramFiles\Tailscale\tailscale.exe" up --login-server={{.ControlURL}} --authkey={{.Key}}

Done — the device will appear on the My devices page in a few seconds.

Want to sign in with a different account?

By default the key is bound to {{.Username}}. If this device is already signed in to Tailscale under a different account, log out first: 

tailscale logout

Then re-run tailscale up --login-server={{.ControlURL}} --authkey={{.Key}}. On mobile (Android/iOS) — log out via → Sign out, then repeat the steps above.
