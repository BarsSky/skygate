Android / iOS
  1. Install Tailscale (Google Play / App Store).
  2. In the app: Settings → Use custom coordination server → paste the URL above.
  3. Tap Sign in → on the login page pick "Use a preauth key" → paste the key.
Linux / macOS / Windows (CLI) 
sudo tailscale up --login-server={{.ControlURL}} --authkey=YOUR_KEY

Add --accept-routes if you want to accept routes from exit nodes.
