Подключить новое устройство

Выберите ОС — мы сгенерируем одноразовый ключ, действующий 1 час. Ключ будет привязан к вашему аккаунту {{.Username}}.

Как подключить (краткая шпаргалка)

Наш tailnet работает на собственном headscale. Перед вводом ключа нужно указать адрес нашего control server:

{{.ControlURL}}
Android / iOS — включить custom server и ввести ключ
  1. Установите Tailscale (Google Play / App Store).
  2. В приложении: Settings → Use custom coordination server → вставьте адрес выше.
  3. Нажмите Sign in → на странице входа выберите "Use a preauth key" → вставьте полученный ключ.
Linux / macOS / Windows (CLI) — одна команда 
sudo tailscale up --login-server={{.ControlURL}} --authkey=ВАШ_КЛЮЧ

Добавьте --accept-routes если хотите принимать маршруты от exit nodes. Полная инструкция с GUI-вариантами появится на следующей странице после нажатия «Получить ключ».

Уже вошли под другим аккаунтом?

Сначала выйдите: tailscale logout (на мобильных — → Sign out). Затем повторите команду tailscale up с новым ключом.
