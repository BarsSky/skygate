Рекомендации

  • 🟢 Зелёный (<50%): норма. Добавляйте правила свободно.
  • 🟡 Жёлтый (50-80%): средняя нагрузка. Следите за latency sync.
  • 🔴 Красный (>80%): высокая нагрузка. Рекомендуется вынести часть правил на другой exit-node или уменьшить количество доменов (autoupdater резолвит много IP).
  • ⏱️ Staggered sync: autoupdater обновляет IP батчами (SKYGATE_STAGGER_BATCH_SIZE) с интервалом (SKYGATE_STAGGER_INTERVAL) — не перегружает headscale approve-routes.
  • 🔄 Last sync: последний успешный sync для этого узла. Если «никогда» — autoupdater ещё не дошёл.
