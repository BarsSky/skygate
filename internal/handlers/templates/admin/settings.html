Exit Node Policy

Allow All: весь трафик через exit-ноды разрешён. Правила действуют как чёрный список.
Deny All: трафик через exit-ноды запрещён. Правила действуют как белый список.

После миграции

Если вы восстановили бэкап на новом сервере:

  1. Адрес Headscale внутри Docker сети (обычно headscale) — может отличаться
  2. Публичный домен изменится — обновите выше
  3. API ключ headscale нужно создать заново на новом сервере: docker exec headscale headscale apikeys create
  4. После сохранения настроек — перезапустите skygate
  5. Проверьте Бэкапы → Тесты для верификации
